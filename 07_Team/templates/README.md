# Project management templates

Reusable templates for running a ShotFlow-style AIGC short-film project. Each template
has an English primary version and a Chinese sidecar (`.zh.md`).

| Template | English (primary) | Chinese |
|----------|-------------------|---------|
| Project proposal | [project_proposal.md](./project_proposal.md) | [project_proposal.zh.md](./project_proposal.zh.md) |
| Progress checklist | [progress_checklist.md](./progress_checklist.md) | [progress_checklist.zh.md](./progress_checklist.zh.md) |
| Instructor review | [instructor_review_template.md](./instructor_review_template.md) | [instructor_review_template.zh.md](./instructor_review_template.zh.md) |
| Weekly report | [weekly_report_template.md](./weekly_report_template.md) | [weekly_report_template.zh.md](./weekly_report_template.zh.md) |
| Summary report | [summary_report_template.md](./summary_report_template.md) | [summary_report_template.zh.md](./summary_report_template.zh.md) |

## Status

All five templates are fully bilingual — both the English and Chinese versions contain
the complete content, with a language-switch link at the top of each file. See
[docs/i18n/README.md](../../docs/i18n/README.md) for the contributor flow.
