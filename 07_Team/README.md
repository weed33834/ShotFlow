# 07_Team — Team and process

Team roles, task assignments, and project-management templates. The templates are
reusable for any ShotFlow-style AIGC short-film project.

## Layout

- [`expert_team.md`](./expert_team.md) — core team roles and responsibilities.
- [`roles.md`](./roles.md) — role definitions and decision authority.
- [`collaboration_tools_guide.md`](./collaboration_tools_guide.md) — team collaboration
  tools setup guide (chat, kanban, docs, storage).
- [`templates/`](./templates/) — reusable project-management templates (proposal,
  progress checklist, instructor review, weekly report, summary report). See
  [`templates/README.md`](./templates/README.md).
- `daily_briefs/` — daily stand-up briefs output by `08_Automation/daily_brief.py`
  (generated, gitignored).

## Templates

The [`templates/`](./templates/) directory holds the five project-management templates.
Each has an English primary stub and a Chinese sidecar (`.zh.md`) with full content.
