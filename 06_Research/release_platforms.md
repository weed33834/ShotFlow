## 一、发布目标

### 1.1 总目标

1. **正片全平台上线**：将《奇点回响》正片及衍生内容（竖版剪辑、幕后图文、技术拆解）同步分发至 9 类平台，覆盖影视受众、AIGC 技术社区与泛科技用户三类核心人群。
2. **技术社区工作流教程发布**：将 ComfyUI 工作流 JSON、SOP 操作手册、参数调优记录以开源形式发布至 GitHub / HuggingFace / Civitai，并配套发布知乎深度文章与 B 站教程视频，形成"作品 + 方法论"双轮驱动的传播闭环。

### 1.2 量化目标

| 维度 | 目标 |
|------|------|
| 全平台累计播放量 | ≥ 100,000 次 |
| 技术社区开源仓库 Star | ≥ 200 |

| 媒体/社区自发转载 | ≥ 10 家 |

## 四、封面与缩略图规范

### 4.1 通用规范

| 项目 | 要求 |
|------|------|
| 主体 | 选取高对比、强视觉冲击的关键帧（角色特写 / 场景奇观） |
| 构图 | 主体居中或黄金分割点，留出文字区域 |
| 文字 | 标题文字不超过 12 字，字号占画面高度 ≥ 1/8，使用无衬线粗体 |
| 配色 | 深色背景 + 高饱和强调色（青/橙/品红），契合科幻调性 |
| 角标 | 左上角统一加"4K AI 科幻短片"或"AI MADE"角标 |
| 字体 | 思源黑体 / 阿里巴巴普惠体（商用免费） |
| 禁止 | 不使用低分辨率素材、不堆砌文字、不使用与内容无关的诱导图 |
