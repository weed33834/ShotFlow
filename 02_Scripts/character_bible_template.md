# 角色圣经模板

> 以女主角艾娃为例，供项目角色设计使用。
>
> 本文件为可复用模板，以《奇点回响》为示例，实际项目请按真实角色替换。

## 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 艾娃（Ava） |
| 年龄 | 28 岁 |
| 身份 | 前星际考古学家，现废墟独行者 |
| 身高/体型 | 168cm，偏瘦但肌肉线条明显 |
| 时代/世界观 | 近未来后启示录科幻，地球文明崩塌后的废墟时代 |

## 面部特征（不可变锚点）

| 项目 | 描述 |
|------|------|
| 脸型 | 鹅蛋脸，下颌线清晰 |
| 眼睛 | 琥珀色瞳孔，眼尾微微下垂，眼神疏离但坚定 |
| 眉毛 | 眉峰略高，眉尾细长 |
| 鼻子 | 直鼻，鼻梁纤细 |
| 嘴唇 | 薄唇，自然闭合时常带一丝紧绷 |
| 发型 | 深色短发，略凌乱，左侧别在耳后 |
| 特殊标记 | 右眼下方有一道浅疤；颈后有神经接口发光纹路 |

## 服装锚点（每镜必须保留）

| 项目 | 描述 |
|------|------|
| 外套 | 深灰色防风夹克，右肩有磨损补丁 |
| 内搭 | 黑色高领紧身衣 |
| 下装 | 深色工装裤，膝盖处加固 |
| 靴子 | 磨损的军靴 |
| 配饰 | 腰间挂着旧式数据终端，左手腕有发光手环 |
| 颜色主调 | 黑、灰、暗橙点缀 |

## 性格关键词

- 孤独、坚韧、好奇、内敛
- 动作习惯：常低头沉思、用手环扫描环境、走路时手扶腰间终端

## 表情与姿态参考

| 情绪 | 表情 | 姿态 |
|------|------|------|
| 迷茫 | 眼神放空，嘴唇微张 | 站立不动，手握终端 |
| 警惕 | 眯眼，嘴角下沉 | 半蹲，手扶腰间 |
| 决心 | 下巴微抬，眼神聚焦 | 大步前行 |
| 悲伤 | 眼帘低垂，呼吸沉重 | 背靠墙壁，肩膀微缩 |
