## 每个角色的标准资产

| 类别 | 最少数量 | 用途 |
|------|----------|------|
| 正面肖像 | 1 张 | 全局身份锚点 |
| 侧面照 | 1 张 | 轮廓与发型一致性 |
| 背面照 | 1 张 | 服装/特殊装置一致性 |
| 三视图合成 | 1 张 | IPAdapter / PuLID 全局参考 |
| 表情变体 | 3–5 张 | 特写镜头辅助参考 |
| 服装/姿态变体 | 2–3 张 | 动作场景参考 |

## 角色一致性检查清单

- [ ] 面部特征固定（脸型、五官、发型、瞳色、特殊标记）
- [ ] 服装锚点完整（材质、颜色、配饰、破损位置）
- [ ] 不可变特征已文档化（如疤痕、接口、纹身）
- [ ] 已生成三视图或多角度合成图
- [ ] 已通过盲测或自动相似度检测
- [ ] 盲测分数 ≥ 4.5/5.0 或相似度 > 95%

## 使用建议

1. **全局参考**：在 IPAdapter / PuLID 节点中使用 `Turnaround/` 下的合成图。
2. **特写辅助**：按情绪选择 `Expressions/` 下的对应表情图。
3. **动作场景**：使用 `Outfits/` 下与镜头姿态最接近的参考。
4. **版本控制**：每次调整角色锚点后，版本号 +1，并记录变更原因。
