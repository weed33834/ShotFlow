## 三、关键混音决策

| 时间码 | 内容 | 处理方式 |
|--------|------|----------|
| 00:00–00:02 | 片头黑场风声 | 从 -∞ 渐入至 -30 LUFS |
| 00:52 | 飞船首次出现 | 低频轰鸣 + 弦乐高潮齐奏 |
| 01:48 | 核心 Chamber 揭示 | 音乐骤停 0.5s，仅留核心嗡鸣 |
| 02:56 | 记忆碎片闪回 | 加入反向混响、延迟效果 |
| 03:34 | 选择时刻 | 配乐重新进入，对白清晰突出 |
| 04:16 | 片尾黑场 | 音乐淡出 2s，最后留 0.5s 静默 |

## 四、对白处理

| 参数 | 设置 |
|------|------|
| EQ | 高通 80Hz，轻微提升 3kHz 清晰度 |
| 压缩 | 3:1，阈值 -18dB，快速释放 |
| 去噪 | -18dB 环境底噪抑制 |
| 去齿音 | 阈值 -20dB |

## 五、音效处理

| 类别 | 处理方式 |
|------|----------|
| 环境音 | 低通 + 轻微混响，铺底不抢戏 |
| 机械/科技音 | 中高频增强，加入短延迟 |
| UI 提示音 | 清脆短促，限制在 0.5s 内 |
| 爆炸/光芒扩散 | 低频饱和 + 侧链压缩避让对白 |
