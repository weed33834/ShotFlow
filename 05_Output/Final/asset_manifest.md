# Asset Manifest — *Echo of the Singularity*

> Project: ShotFlow / *Echo of the Singularity* (example)
> Manifest version: v1.0
> Date: 2026-07-04
> License: CNCL v1.0 (see repo root `LICENSE`)

A complete inventory of every asset that goes into the final master. Each row
maps a logical asset (shot, line, cue, keyframe) to its source file and a
checksum. Use this manifest to verify a delivery package is complete and
unmodified.

> **Note**: The actual media files (clips, WAVs, keyframes) are not committed
> to this repo — they are too large and several are generated by NC-licensed
> models. The paths below are the canonical locations to drop them in. Run
> `08_Automation/render_queue.py --manifest` to populate checksums
> automatically once the files exist locally.

## 7. Checksum table

Populate this when packing a delivery. Algorithm: SHA-256.

```text
file                                         sha256
SF_S01_01_WanT2V_v01.mp4                     <fill>
SF_S01_02_Wan_v01.mp4                        <fill>
...                                          ...
Ava_S02_05_v1.wav                            <fill>
...
Main_Theme_v1.wav                            <fill>
...
ShotFlow_4K_Master_v10.mp4                   <fill>
echo_of_singularity.zh.srt                   <fill>
echo_of_singularity.en.srt                   <fill>
```

To regenerate automatically:

```bash
cd 05_Output
sha256sum Renders/*.mp4 Final/*.mp4 Final/*.mov Final/*.wav \
          Final/subtitles/*.srt Final/subtitles/*.ass \
          > Final/checksums.sha256
```

Verify with `sha256sum -c Final/checksums.sha256`.
