# 04_SOP — Standard operating procedures

Manuals and production standards covering the full pipeline. Each SOP is a standalone
reference; together they form the operation handbook referenced by
[`AIGC_Experience_Chain.md`](../AIGC_Experience_Chain.md).

## Files

- [`sop_shotflow.md`](./sop_shotflow.md) — full pipeline SOP: setup, asset generation,
  motion generation, post-production, delivery. The canonical manual.
- [`postproduction.md`](./postproduction.md) — DaVinci edit, color grade (Teal & Orange),
  Topaz 4K upscale, mix notes.
- [`audio_production.md`](./audio_production.md) — dialogue (ElevenLabs), SFX, score
  (Suno/Udio), −14 LUFS mix standard, delivery format.

## Usage

New contributors should read `sop_shotflow.md` end-to-end before touching the pipeline.
The post/audio SOPs are referenced per-phase during weeks 5–6 of the schedule in
[`07_Team/templates/project_proposal.zh.md`](../07_Team/templates/project_proposal.zh.md).
