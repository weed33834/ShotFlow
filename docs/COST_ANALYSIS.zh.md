# ShotFlow — 费用参考

[English](./COST_ANALYSIS.md) | 中文（当前）

> 本文档估算使用本项目制作一部 3–5 分钟 AIGC 短片的两种方案成本：本地 GPU 方案 与 云端 API 方案。实际费用因模型版本、生成次数、重跑次数而异。

---

## 方案一：本地 GPU 方案

适合有 RTX 4090 或同等显卡、希望完全掌控生成过程的团队。

| 项目 | 估算成本 | 说明 |
|------|----------|------|
| RTX 4090 24GB | 已购置约 0 元/租赁约 300–600 元/月 | 若已有设备则成本为 0 |
| 电费 | 约 50–100 元/部 | 按 100 小时满载估算 |
| 模型下载 | 0 元 | 开源模型 |
| 可灵 API（复杂镜头）| 约 50–100 元 | 5 个复杂镜头，按 0.5–1 元/5s 估算 |
| ElevenLabs 配音 | 约 0–50 元 | 免费额度通常足够 |
| Suno 配乐 | 约 0–100 元 | 免费额度可生成多首 |
| 达芬奇/Topaz | 0 元/订阅费 | 达芬奇免费版够用；Topaz 按需订阅 |
| **合计** | **约 100–900 元/部** | 主要取决于是否有本地显卡 |

---

## 方案二：云端 API 方案

适合无本地 GPU、希望快速出片的团队。

| 项目 | 估算成本 | 说明 |
|------|----------|------|
| 关键帧生成（Flux API）| 约 30–60 元 | 29 张关键帧，重跑后约 60–120 张 |
| 标准镜头视频（可灵/Runway）| 约 300–800 元 | 19 个标准镜头，每个 3–5s |
| 复杂镜头视频（可灵首尾帧）| 约 50–150 元 | 5 个复杂镜头 |
| ElevenLabs 配音 | 约 0–50 元 | |
| Suno 配乐 | 约 0–100 元 | |
| 达芬奇/Topaz | 0 元/订阅费 | |
| **合计** | **约 400–1200 元/部** | 取决于 API 选择和重跑次数 |

---

## 省钱建议

1. **关键帧先用本地 Flux 跑**，云端只用来生成视频；
2. **标准镜头优先用 Wan2.2 本地生成**，只有复杂/转场镜头调用可灵；
3. **配音和配乐先用免费额度**，不够再付费；
4. **建立渲染队列**，避免重复生成；
5. **用 `video_quality_check.py` 提前筛掉崩坏镜头**，减少无效重跑。

## 时间成本

| 阶段 | 估算时间 |
|------|----------|
| 策划与分镜 | 3–5 天 |
| 角色一致性验证 | 2–3 天 |
| 关键帧生成 | 1–2 天 |
| 视频生成 | 3–5 天 |
| 后期合成 | 3–5 天 |
| **总计** | **约 2–3 周** |

> 以上估算仅供参考，实际费用以各平台实时定价为准。
